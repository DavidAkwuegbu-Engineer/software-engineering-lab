# Project Ladder

The lab should build skill through projects from the start. Each project should be small enough to understand fully, but structured enough to become credible portfolio evidence.

## Project 1: Developer Profile CLI

Build a Python command-line app that creates, validates, saves, and displays a developer profile.

Core skills:

- Functions and modules.
- Validation of untrusted input.
- Lists, dictionaries, and simple data modeling.
- JSON file handling.
- Command-line interfaces.
- Unit tests.
- Git commits and GitHub evidence.

Portfolio story:

> I built a tested Python CLI that validates developer profile data, stores it safely as JSON, and demonstrates secure input handling and clean project structure.

## Project 2: Developer Profile API

Rebuild the same domain as a small API.

Core skills:

- HTTP routes.
- Request validation.
- JSON responses.
- Error handling.
- API tests.

## Project 3: Developer Profile Dashboard

Build a frontend for viewing and editing the profile.

Core skills:

- React components.
- Forms.
- State.
- Accessibility.
- User experience.

## Project 4: Full-Stack Portfolio System

Turn the project into a deployed full-stack app.

Core skills:

- Next.js.
- Authentication.
- Database storage.
- Deployment.
- Production readiness.

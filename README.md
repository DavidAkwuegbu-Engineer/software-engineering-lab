# Software Engineering Lab

This repository is a public learning lab for building software engineering skill through deliberate practice.

The goal is to become a strong, security-minded software engineer across Python, JavaScript, TypeScript, frontend engineering, full-stack systems, databases, APIs, applied AI, testing, debugging, and deployment.

## How This Repo Is Structured

- `00-curriculum`: learning plans, standards, and workflow notes.
- `01-python-fundamentals`: Python exercises and test-driven practice.
- `02-javascript-typescript`: JavaScript and TypeScript practice.
- `03-react-frontend`: frontend application work.
- `04-nextjs-fullstack`: full-stack application work.
- `05-supabase-postgres`: database and backend data work.
- `06-apis-auth-realtime`: API, auth, and realtime systems.
- `07-applied-ai`: applied AI product engineering.
- `08-testing-debugging-deployment`: production-quality verification and release practice.
- `portfolio-evidence`: short evidence notes for completed work.

## Engineering Standards

- Write clear, readable code.
- Validate untrusted input.
- Prefer secure defaults.
- Add tests or checks for meaningful behavior.
- Keep commit history understandable.
- Document what was built and how it was verified.
